# Deepfake Voice Recognition Web App

This application downloads a deepfake voice dataset from Kaggle, trains a Random Forest Classifier using acoustic MFCC features, and deploys an interactive interface via Streamlit.

## Target Output Specs
* **REAL**: Authentic human speech.
* **FAKE**: AI-generated synthetic clone audio.
